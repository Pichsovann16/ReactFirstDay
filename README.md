# Movie DB (React)

[View the site]()

## Features
- Movie list (popular movies) with poster, title, release date, rating
- Live search (filters by title as you type)
- Movie details page (large poster, title, rating, overview, genres)
- Responsive grid UI (mobile friendly)

## Setup (TMDB API)
1. Create a TMDB API key.
2. Create a `.env` file (same folder as `package.json`) and add:

   REACT_APP_TMDB_API_KEY=your_tmdb_api_key_here

   (You can copy `.env.example`.)
3. Restart the dev server after adding/changing env vars.
